# Définition de macros

Projet    : \sigle  
Auteur    : \author  
Version   : \version

> Vérifie que les macros fonctionnent aussi :

> - en fin de ligne : \sigle

> - à l’intérieur d’un mot : pré\sigle-post

> - plusieurs fois dans la même phrase : \sigle, \sigle, \sigle  

Test d’une macro non définie pour l’erreur : \undefinedMacro
